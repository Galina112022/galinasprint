Проект «Разработка темы интернет-магазинов для Wordpress»

Концепция проекта

  1. Миссия проекта

    2. Предложить достойный продукт пользователям, аналог WoodMart.

  2. Цель проекта

    3. В рамках обучения разработать аналог WoodMart. Разработать сайт, на котором информативно представить все преимущества разработанного продукта для потенциальных покупателей.

  3. Идея проекта

    4. Разработанная тема должна быть достойным аналогом зарубежного сайта WoodMart, покупка последнего в России более не доступна. Тема будет отвечать современным требованиям к интернет-магазинам и полностью адаптивная. Тема полностью адаптивна и поставляется с живым настройщиком WordPress. С расширениями WooCommerce и виджетами WooCommerce тема проста и легко настраивается. Он предлагает раздел дочерних тем, чтобы была возможность изменять общие шаблоны. Тема, подкрепленная строгими проверками безопасности, работает очень быстро, чтобы сделать работу более продуктивной. Она поддерживает все типы устройств и может быть переведен на любой из языков для удобства в работе.

Заинтересованные лица (стейкхолдеры)

  1. Команда обучающихся на курсах 1Т Спринт;
  2. Корпорация 1Т;
  3. Потенциальные клиенты: разработчик сайта (он может быть пользователем, когда работает один или в небольшой команде), владелец (заказчик) сайта, аналитик, который ищет подходящую тему для сайта заказчика, web дизайнер, который так же ищет подходящую тему для сайта заказчика.
